# Facebook Python Toolkit

Python scripts for Facebook automation: group management, invites, data dumping.

## When to use
- Facebook group operations (invite, leave, find)
- Data extraction from Facebook
- Bulk Facebook operations

## Scripts
- dump_fb.py — Extract Facebook data
- group_invite.py — Send group invitations
- group_leav.py — Leave groups

## Requirements
- Python 3.x
- Facebook session cookies or API access
