## ✨ Features

> All features below are based on files that **actually exist** in this repository.

| Script | What It Does |
|---|---|
| 🗑️ `post_delete.py` | Bulk-delete posts from Facebook groups you manage |
| 🚪 `group_leav.py` | Leave multiple Facebook groups automatically |
| 📨 `group_invite.py` | Auto-invite users to a Facebook group |
| ❌ `spam_decline.py` | Auto-decline spam group join requests |
| 📦 `dump_fb.py` | Scrape and export Facebook data for analysis |
| 🔍 `mabsic group finder/` | Search and discover Facebook groups by keyword |

## 🛠️ Installation

Works on **Windows**, **macOS**, and **Linux**.

### 1. Clone the repo

```bash
git clone https://github.com/Mahdi-hasan-shuvo/Facebook-automation.git
cd Facebook-automation
```

### 2. Create a virtual environment *(recommended)*

```bash
# Create
python -m venv venv

# Activate — Windows:
venv\Scripts\activate

# Activate — macOS / Linux:
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install selenium beautifulsoup4 requests
```

### 4. Set up ChromeDriver

**Option A — Automatic (easiest):**

```bash
pip install webdriver-manager
```

Then in your script, replace the driver setup with:

```python
from selenium import webdriver
from webdriver_manager.chrome import ChromeDriverManager

driver = webdriver.Chrome(ChromeDriverManager().install())
```

**Option B — Manual:**

1. Check your Chrome version: `chrome://settings/help`
2. Download the matching ChromeDriver from [chromedriver.chromium.org](https://chromedriver.chromium.org/downloads)
3. Place it in your project folder or add it to your system PATH

## ⚙️ Configuration

> **Note:** The repo does not yet include a `.env` file. Credentials are currently set directly inside each script.

### Option A — Edit the script directly

Open any script and look for the config section near the top:

```python
# ── CONFIG ──────────────────────────────────
EMAIL     = "your_email@example.com"
PASSWORD  = "your_password"
GROUP_URL = "https://www.facebook.com/groups/your-group-id"
# ────────────────────────────────────────────
```

Replace the values and save.

### Option B — Use a `.env` file *(recommended)*

Create a `.env` file in the project root:

```ini
FB_EMAIL=your_email@example.com
FB_PASSWORD=your_password
TARGET_GROUP=https://www.facebook.com/groups/your-group-id
```

Install `python-dotenv`:

```bash
pip install python-dotenv
```

Load it in your script:

```python
from dotenv import load_dotenv
import os

load_dotenv()
email     = os.getenv("FB_EMAIL")
password  = os.getenv("FB_PASSWORD")
group_url = os.getenv("TARGET_GROUP")
```

> 🔒 **Always add `.env` to `.gitignore`** — never commit your password to GitHub.

## ▶️ Usage

Run any script individually from your terminal:

```bash
# Delete all posts in a group
python post_delete.py

# Leave groups in bulk
python group_leav.py

# Invite users to a group
python group_invite.py

# Decline spam join requests
python spam_decline.py

# Scrape and export Facebook data
python dump_fb.py

# Find Facebook groups by keyword
cd "mabsic group finder"
python main.py
```

### Example output

```

[INFO] Logging in to Facebook...
[INFO] Login successful ✓

...
[INFO] All done! 52 posts deleted.
```

> 🔁 Scripts run **continuously** until the task is fully complete — perfect for large groups.

## ⚠️ Safety & Responsible Use

Please read before running any script.

- 📖 Automating Facebook actions may violate [Facebook's Terms of Service](https://www.facebook.com/terms.php). Your account could be restricted or banned.
- 👤 Only use these scripts on **accounts you own**.
- 🐌 Do not remove built-in delays — they reduce the risk of triggering bot detection.
- 🔒 Never commit your Facebook credentials to a public repository.
