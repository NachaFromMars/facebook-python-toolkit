# facebook-python-toolkit — Python scripts for Facebook group automation

> A small collection of Python utilities for Facebook group operations — data extraction, group invitations, and bulk leave actions.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
facebook-python-toolkit provides focused Python scripts for automating Facebook group tasks. Each script handles a specific operation: extracting Facebook data, sending group invitations, or leaving groups in bulk. Requires Python 3.x and Facebook session cookies or API access.

## Features
- **`dump_fb.py`** — extract Facebook data
- **`group_invite.py`** — send group invitations
- **`group_leav.py`** — leave groups (bulk)

## Usage / Quick Start
```bash
# Extract data
python3 dump_fb.py

# Send group invitations
python3 group_invite.py

# Leave groups
python3 group_leav.py
```
Requires: Python 3.x + Facebook session cookies or API access.
